ООО «КНВ» - разработчик платформы для операторов связи по предоставлению виртуальных АТС по модели SaaS.

ООО «КНВ» является разработчиком платформы для операторов связи по предоставлению функциональных и легко управляемых виртуальных АТС своим абонентам под собственным брендом оператора связи (white label). Предоставление осуществляется по модели SaaS.

  • Технологический стек платформы Gurdy:
  • ОС Debian Linux
  • Docker
  • СУБД PostgreSQL
  • http-сервер Nginx
  • Kamailio
  • FreeSwitch
  • Используемые языки программирования:
  • javaScript
  • Lua
  • Golang
